# Lab 7 - Laplacian, Mathematical, and Statistical Filters

This project contains Python implementations for **Lab 7** in Image Processing.  
The lab is divided into two main parts:

- **Part A:** Laplacian Filter
- **Part B:** Mathematical and Statistical Filters

The goal of this lab is to apply different filtering techniques to grayscale images for:

- image sharpening
- noise reduction
- salt-and-pepper noise removal
- smoothing and enhancement
