# 🧪 Lab 6 – Frequency Domain Filtering  
**Course:** ARTI404 – Image Processing  
**University:** Imam Abdulrahman Bin Faisal University  

---

## 📌 Overview
In this lab, we applied image processing techniques in the **frequency domain** using Python.  
The goal is to understand how filters can be applied using **Fourier Transform (FFT)** instead of spatial convolution.

---

## 🎯 Objectives
- Apply **Laplacian filter** in the frequency domain  
- Apply **Sobel filters (X & Y)** in the frequency domain  
- Detect edges using frequency-based filtering  
- Visualize results (spectrum + output images)
